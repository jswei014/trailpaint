Spot popup 配圖：每個 spot 的 `photo` 欄位塞入 400px JPEG base64（quality 75），段內 2 張畫作輪替。共 24 spot × ~30 KB ≈ 720 KB 分散於 3 個 .trailpaint.json。

## 處理流程（reproducibility）

1. 使用 `curl` 透過 Wikimedia `Special:FilePath` API 下載 `?width=800` 版本
2. Python PIL resize 到最長邊 400px（spot popup）/ 400px (thumb) / 800px (cover) / 1200×630 (og center crop)
3. JPEG quality：spot popup = 75、thumb/cover = 82、og = 82
4. 處理腳本：`/tmp/process_passion_imgs.py`（保留供未來替換或重新生成）
